《Cetaは Borland C++ Builder 4.0 のbcc32コンパイラで作製されました。》

《紹介と特徴》
・本来Windowsプログラムの実験として作製されたものです。適度に肉付けを
し、一つのまともなプログラムとして完成させました。
・カラー指定の時の名称がwinuser.hで使われているものそのままなので、
ある程度プログラムをたしなまれる方でないと使うのは難しいでしょう。おい
おい改良したいと思います。
・よーするに、「画面のプロパティ」->「デザイン」を自作したものです。
ただし、こちらの方がより細かい色指定が可能です。
・レジストリに書き込ま無いので、電源を切れば元に戻ります。
・設定に名前を付けてファイルに書き出すことができます。逆に、読み出す
ことも可能です。ファイルの拡張子は「*.skn」です。

《使用条件》
　本プログラムを実行した事により発生したいかなる動作にも、たとえ原因が
本プログラムにあるとはっきりした場合でも、一切の責任を作者の風晶は負わ
ないものとします。
　
《使い方》
・本プログラムにはヘルプがありません。使い方は簡単ですので、ここで説明
します。
１．実行すると、その時点でのデスクトップのカラーデザインが読み込まれます。
２．リストボックスから変更したい部分を選択します。
３．リストボックスの隣のエディットボックスで、RGB値を変更します。
４．「これだけ色替え」ボタンをクリックすると、現在リストボックスで選択
されているカラーが、エディットボックスで設定された色に変更されます。
５．「これだけ戻す」ボタンをクリックすると、現在リストボックスで選択さ
れているカラーが実行前の状態に戻ります。
６．「全部色替え」ボタンをクリックすると、全てのカラーが変更されます。
７．「全部戻す」ボタンをクリックすると、全てのカラーが実行前の状態に
戻ります。
８．「設定を読み込む」ボタンをクリックすると、「開く」ダイアログボッ
クスが表示されます。そこでWindow's Skinner専用の*.skn設定ファイルを
指定すれば、設定が読み込まれます。
９．「設定を書き出す」ボタンをクリックすると、「名前を付けて保存」ダ
イアログボックスが表示されます。そこでファイル名を指定して、現在の設
定をファイルに書き出すことができます。

《転載条件その他》
転載、改造の一切を許可します。

《動作環境》

メモリ、HDD：特に条件無し
OS：Windows98のみ
