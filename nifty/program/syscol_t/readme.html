《紹介と特徴》
　WinAPI32において、GetSysColor()、SetSysColors()で指定する、各種定義済
マクロ。これらのマクロ値が実際にどのウインドウパーツに対応しているのか
確認するための実験プログラムです。
　レジストリは一切いじくりません。万一フリーズしてもリセットすればもと
の状態に戻ります。「全ての設定を元に戻す」ボタンを押せば、数十秒の後、
もとのカラー状態に戻ります。

《転載条件その他》
基本的に転載は自由です。

《動作環境》
特になし。

《使用方法その他》
コンボボックスから指定したいマクロ名を選択して下さい。
R(Red),G(Green),B(Blue)の各エディットボックスにパラメータを指定して下さ
い。
「カラー変更」ボタンを押せば、現在コンボボックスで選択されているマクロ
の色を、R,G,Bエディットボックスで指定されているとおりの色に変更します。
設定を元に戻したいときは「全ての設定を元に戻す」ボタンを押して下さい。
２５個ある全ての設定値を戻すので、多少時間がかかります。
