「Ceta(ツェータ)　ご案内」に戻る

　というわけで、よくわからん中途半端この上ないCetaを作者自身はどう使っているか。
　要するにCetaの使い方の実例集ですね。

　いろいろ試して、少しずつ増やして行けたらと思いますが予めこれだけは断っておきます。
　あまり正確さを求めないで下さい。
　どちらかというと、「メモ書き」位に思って参考にしていただければな、と思います。
　ただし、分からない点や疑問等ありましたらいつでもメールにてご相談下さい。

　じゃあ、始めます。


  1. Borland C++ Compiler 5.5を登録してみる。
  2. Borland C++ Compiler 5.5でWindows用オプションを設定する。
  3. Borland C++ Compiler 5.5の「リソースコンパイラ」を登録する。
  4. Borland C++ Compiler 5.5の「ilink32.exe」を登録する。
  5. InternetExplorer を登録して、HTMLファイルを編集してみる。
  6. PICアセンブラを登録してみる。
  7. PICライターを登録してみる。
  8. (何の役に立つんだ)DOSプロンプトを登録してみる。
  9. Java2 SDKを試す（その１）javac（Java コンパイラ）
  10. Java2 SDKを試す（その２）java（Java 実行プログラム）
  11. 「QKC - Quick KANJI code Converter」を利用してShift_JIS←→euc-jp変換にこき使う。
  12. 「Salford Fortran 77」でFortranプログラミング。
