2015-03時点で、jQueryでソートとかグルーピングとか出来ちゃう高機能なTable/Gridライブラリってどんなのがあるんだろうか、といくつかのまとめサイトから好評そうなのを見繕って、参考URLをまとめたメモ書きです。

  • データ操作に役立つjQueryプラグイン （1/4）：CodeZine
    • http://codezine.jp/article/detail/5946
  • グリッドレイアウト採用時に便利なjQueryプラグイン 20 | NxWorld
    • http://www.nxworld.net/tips/jquery-grid-layout-plugins.html
  • JavaScriptのGridライブラリ決定版 - SlickGrid - Qiita
    • http://qiita.com/icoxfog417/items/98e34c0555991033afec
  • テーブルを使ったいろいろなjQueryプラグイン26個まとめ | Web活メモ帳
    • http://blog.verygoodtown.com/2011/05/jquery-table-plugins/

先に結論的なのをまとめると、以下の三点になります。ライブラリ自体の品揃えは豊富なんですが、本当に良いライブラリとなるとお金がかかります。
したがって、Data Table/Gridを必要としているWebシステム全体のライフサイクルや開発体制を考慮して取捨選択する必要があります。

  • 本当に高機能でブラウザ互換性が取れてて開発基盤が安定しているライブラリは、商用利用で有償ライセンスになるというのがデフォ。
  • 個人ベースで開発されてるOSSだと、メイン開発者の個人事情で開発がストップする可能性があるので、選択時にはその点を考慮のこと。
  • 今の時点でOSSであっても、開発側の事情で非商用利用のみフリー、商用だと有料ライセンスになるケースもある点を考慮のこと（jqGrid -> Guriddoの例）。

以下、有償製品 -> OSSの順にめぼしいライブラリのURLとライセンスについて簡単にまとめました。
なお、ライセンス情報や開発状況についての記載は2015-03時点のものになる点に注意して下さい。

jQuery EasyUI(有料ライセンスあり)

  • datagrid - Documentation - jQuery EasyUI
    • http://www.jeasyui.com/documentation/datagrid.php
    • jQuery EasyUI の中のgridコントロール。
    • GPL版か有料の商用利用可バージョンのデュアルライセンス。

IgniteUI(有料ライセンスあり)

  • jQuery コントロール - jQuery コントロールのサンプル - IgniteUI 注目のコントロール
    • http://jp.igniteui.com/
    • IgniteUIの中にGridコントロールあり。
    • IgniteUI自体は有償のUIライブラリ製品。
  • Edit fiddle - JSFiddle
    • http://jsfiddle.net/gh/get/jquery/1.9.1/igniteuisamples/jsfiddle-samples/tree/master/JA/HtmlSamples/grid/json-binding/
    • IgniteUIのGridコントロールのサンプル

jQWidgets(有料ライセンスあり)

  • jQuery Grid, JavaScript Grid, jqxGrid, DataGrid, jQuery DataGrid
    • http://www.jqwidgets.com/jquery-widgets-demo/demos/jqxgrid/
    • jQWidgetsの高機能なGridコントロール。
    • Non-Commercialなら Creative Commons Attribution-NonCommercial 3.0 License, 商用利用では有料となる。

DataTables(サポートが欲しい人向けの有償ライセンス有り)

  • DataTables | Table plug-in for jQuery
    • http://www.datatables.net/
    • jQueryプラグインとして開発されたGrid/Tableコンポーネント。
    • MITライセンスで商用利用可。サポートが必要であれば有償ライセンスの購入になる。
    • スコットランドを拠点とする SpryMedia 社が開発している。

jqGrid -> Guriddo (有料ライセンスあり)

  • jQuery Grid Plugin – jqGrid
    • http://www.trirand.com/blog/
    • https://github.com/tonytomov/jqGrid
    • http://www.trirand.com/jqgridwiki/doku.php
    • Grid系のjQueryプラグインとしては老舗。
    • MITライセンスで商用利用可

→利用者の増加とともに要望も増え、より開発力を増すため、非商用は Creative Commons Attribution-Non-Commercial 3.0 License で、商用利用なら有料ライセンスというデュアルライセンスに切り替えた。
Guriddoでは、PHPなどサーバサイドとの連携にも注力している。

  • jQuery Grid Plugin – jqGrid » jqGrid is now Guriddo
    • http://www.trirand.com/blog/?p=1438
  • Guriddo
    • http://guriddo.net/
  • Guriddo » Pricing/Purchase Guriddo jqGrid JS
    • http://guriddo.net/?page_id=103334

w2ui

  • Home | JavaScript UI - w2ui
    • http://w2ui.com/web/
    • 高機能なGridコンポーネントを含んだUIライブラリ。
    • MITライセンス
    • 個人ベースで2014年に開発が盛り上がっている。現在も開発は活発なようだ。
  • vitmalina/w2ui
    • https://github.com/vitmalina/w2ui
  • Demos: Grid | JavaScript UI - w2ui
    • http://w2ui.com/web/demo/grid
  • jQuery - 100万行扱えるグリッドw2ui GridとHandsontableの紹介 - Qiita
    • http://qiita.com/hnakamur/items/052c439674fa92848ada

Handsontable

  • handsontable
    • https://github.com/handsontable/handsontable
    • jQueryには依存しておらず、他のライブラリとも組み合わせることが出来る。
    • テーブルを作る、というよりはExcelのUIに特化している。
    • MITライセンス
    • 2015-03現在、数人の開発者からのcommitで活発に開発が続いている。
  • Conditional formatting - Handsontable
    • http://handsontable.com/demo/conditional.html

SlickGrid

  • mleibman/SlickGrid
    • https://github.com/mleibman/SlickGrid
    • jQuery UI上で構築されたgridライブラリ。
    • MITライセンス
    • 2014-05以降、開発者の都合で開発ストップ。2014年になり、リリース無し。
  • Home · mleibman/SlickGrid Wiki
    • https://github.com/mleibman/SlickGrid/wiki
  • SlickGrid example: Optimizing DataView
    • http://mleibman.github.io/SlickGrid/examples/example-optimizing-dataview.html
