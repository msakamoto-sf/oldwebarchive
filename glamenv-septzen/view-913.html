日記/2011/02/01/Firefoxのオフラインモード(Work Offline)を抑制

  • Firefox： オフラインモードのリロード - 檜山正幸のキマイラ飼育記
    • http://d.hatena.ne.jp/m-hiyama/20110201/1296524294

で調べた時のURLメモ集。

最初は「オフラインモード」の事をよくわかってなかったので、「キャッシュをクリア出来れば良いのかな？」と調べてCache-Statusプラグインに到達した。

  • Mozilla Re-Mix: Firefox 3のメモリ/ディスクキャッシュを自動的にクリアしてくれるアドオン「Cache Status」
    • http://mozilla-remix.seesaa.net/article/102710472.html
  • Cannot clear cache - Offline storage button "Clear Now" does not work | Firefox Support Forum | Firefox Help
    • http://support.mozilla.com/en-US/questions/683900
  • Offline resources in Firefox - MDC Doc Center
    • https://developer.mozilla.org/En/Offline_resources_in_Firefox
  • 猫科研究所(felid labo) - Firefoxのcache関係
    • http://www.up-cat.net/Firefox%25A4%25CEcache%25B4%25D8%25B7%25B8.html
  • Cache Status :: Add-ons for Firefox
    • https://addons.mozilla.org/ja/firefox/addon/cache-status/

が、どうやら"File > Work Offline"の事だと自分の勘違いに気づき、調べ直す。

  • Online and offline events - MDC Doc Center
    • https://developer.mozilla.org/en/online_and_offline_events
  • Toolkit.networkmanager.disable - MozillaZine Knowledge Base
    • http://kb.mozillazine.org/Toolkit.networkmanager.disable

"toolkit.networkmanager.disable"を弄れば、オフライン・オンラインの切り替えを抑制できるっぽいが、これLinux環境オンリー。Windowsじゃ無理ぽ。

で、次の記事にぶつかった。

  • Firefox 3.5 Automatically changes to work offline mode
    • http://forum.notebookreview.com/windows-os-software/399522-firefox-3-5-automatically-changes-work-offline-mode.html

ここで"StayOnlineMode"というプラグインが言及されている。これが一番アタリっぽい。

  • StayInOnlineMode Home - Firefox stays in online mode, even if the internet is disconnected
    • http://netticat.ath.cx/StayOnline/StayOnline.htm
  • StayInOnlineMode :: Add-ons for Firefox
    • https://addons.mozilla.org/en-US/firefox/addon/stayinonlinemode/

m-hiyamaさんと似たような不満を感じている人もいるらしい。

  • Firefox opens with "work offline" checked, and I can't get it unchecked | Firefox Support Forum | Firefox Help
    • http://support.mozilla.com/ja/questions/757685

"StayOnlneMode"を試そうと思ったんだが、どうも自分の環境では「自動的にオフラインモードに切り替わる」という現象そのものがまず発生しない。そのため、「オフラインモードへの自動切換えを抑制」する確認がそもそも出来ず、評価できなかった。Cache-Statusの確認では手動でオフラインモードに切り替えたりしてる。

"StayOnlineMode"プラグインはまだまだ発展途上。Firefox自身のOffline/Onlineモード切り替えの変更もあるかもだし、将来は未知数。

以上、メモ。
