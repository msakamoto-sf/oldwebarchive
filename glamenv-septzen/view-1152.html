EclipseでGroovy/Grailsを楽しむためのメモ。

2013-04-22 update : Groovy/Maven/Examples (includes "Java Joint Compile") により深く、Maven + GMaven 構成について調査・検証していますので、そちらも参照してください。本記事で紹介・参照している GMaven よりもより実際の開発に即した構成を紹介しています。

plugin

  • プラグイン：Eclipse Marketplaceで"GGTS"(Groovy/Grails Tool Suite)を導入するとお手軽。
    • http://grails.org/products/ggts

既存のJavaプロジェクトへの組み込み

  • 既存のJavaのプロジェクトにGroovyを混ぜるには：
    • GGTS入れた後で、JavaプロジェクトでGroovyクラスを新規作成する。Groovyのプラグインが自動的に設定を更新してくれる。
  • 既存のJava + MavenにGroovyを混ぜ込むには：
    • GMavenプラグインを使う。ただし、Groovyのランタイムやコンパイルオプションについて、常に最新に追従して行けている訳ではないので注意。また、個人ベースでメンテナンスされているのでパワー不足の感じも否めない。
    • Eclipse + m2e を既にセットアップ済みであれば、かなりスムーズに Java/Groovy + Maven + Eclipse によるサクサク開発を始められる。
