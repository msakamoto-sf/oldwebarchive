使ってるアプリのメモ。2012～2013頃に揃えたので、もしかしたら最近だとインストールできなかったり公開やめたのもあるかも。

ブラウザ

  • Firefox
  • Google Chrome

エディタ

  • mi
  • ターミナルソフトからvim起動してそれで済ませがち。
  • 0xED (バイナリエディタ)
  • Synalize it! (バイナリエディタ)

日本語入力

  • Google 日本語入力

ネットワーク系

  • Cyberduck : FTP, SFTPなどのファイル転送に使った。
  • Wireshark : パケットキャプチャ
  • uTorrent : BitTorrentの軽量クライアント

ファイルの圧縮と解凍

  • DropLha, DropUnLha
  • WinArchiver Lite

AVプレイヤー

  • Elmedia Player : wmv/mp4/flv と、主要な動画ファイルに対応してくれてる。

その他

  • AppCleaner : アプリのアンインストールや、ゴミファイルの削除に。
  • JD-GUI : Javaのデコンパイラ。Win版もある。フリーソフト。
  • SourceTree : Atlassianから無料で提供されてるGitのGUIクライアント。
  • KeePassX : パスワードマネージャ
  • SimpleCap : スクリーンショットツール。画面キャプチャ。
  • iConvert icons : アイコン編集するときに使った。
  • CHM Reader Plus : WindowsのCHM形式ヘルプファイルのビューア。有料、購入した。
  • Dropbox
  • Kindle
  • Skype
  • Witch : ウインドウ切り替え。デフォのcommand + tabではアプリ切り替えになり、ウインドウ単位で切り替わらなかったので、これを購入して、option + tab でウインドウ単位で切り替えるようになった。
