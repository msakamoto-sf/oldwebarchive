とりあえず面白そう、使ってみようかな～と思ったアプリのメモ。インストールせず放置したり、しても使ってみてイマイチだったりしたのもゴチャマゼに置いてあります。一口メモ的な感想は完全に個人の独断と偏見に基づいてるので、あまり参考にしないでください。自分が参考にしてるだけです。

  • 開発者とかリバースエンジニアリングとか、そっち系
  • UnixとかLinuxのリファレンスマニュアル持ち歩けたら便利そう？
  • 端末(Terminal Emulator)ソフトとかあるかな？

開発者とかリバースエンジニアリングとか、そっち系

  • Manual Intent Shortcuts - Android マーケット
    • https://market.android.com/details?id=net.cwfk.ig88.intents
  • GetIntentList - Android マーケット
    • https://market.android.com/details?id=yanzm.example.getintentlist
  • Package Explorer - Android マーケット
    • https://market.android.com/details?id=org.andr.pkgexp
  • Android Intent Playground - Android マーケット
    • https://market.android.com/details?id=com.codtech.android.intentplayground

UnixとかLinuxのリファレンスマニュアル持ち歩けたら便利そう？

  • Unix & Linux Commands - Android マーケット
    • https://market.android.com/details?id=com.jpsecurenet.unixlinuxcommands
  • UnixRef - Android マーケット
    • https://market.android.com/details?id=jean.unix
  • Linux Commands - Android マーケット
    • https://market.android.com/details?id=com.dekryptedit.LinuxCommands
  • Linux Command List - Android マーケット
    • https://market.android.com/details?id=com.application.LinuxCommandList

全体として、特に嬉しい訳でもないし嬉しくない訳でもない、ビミョーなカテゴリ。
今操作しているUnixのmanやinfoを見たい、という状況が多いので、だったらすなおにmanやinfoコマンド叩くよね、わざわざLinuxとかUnixの特定のバージョンしか対象としていないリファレンスマニュアルを、小さなAndroid端末の画面で目を凝らして読みたくないなぁ、というのが正直な感想。

あと、広告表示するためだけにインターネットアクセス要求してるのがウザイ。

端末(Terminal Emulator)ソフトとかあるかな？

  • Android Terminal Emulator - Android マーケット
    • https://market.android.com/details?id=jackpal.androidterm
      • SSHというよりは、Android自身のターミナルを起動する。
      • ぶっちゃけAndroid SDK入れて "adb shell" 叩ければ不要かも。
