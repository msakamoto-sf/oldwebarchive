メモ段階です。未検証。Jenkinsと組み合わせる場合は 日記/2012/06/10/Jenkinsのビルド情報をMANIFEST.MFに埋め込む 参照。

ソースツリーにあるカウンタファイルあるいは日時などからビルド番号を埋め込むタイプ：

  • Build Number Maven Plugin - Introduction
    • http://mojo.codehaus.org/buildnumber-maven-plugin/index.html
    • システムにインストールされているsvnコマンドをデフォルトでは利用。
    • Windowsなどsvnコマンドが無い環境では、 maven-scm-provider-svnjava 経由でsvnkitを使ってjavaの世界だけでsvnの情報を取得するよう調整可能。
      • see: http://mojo.codehaus.org/buildnumber-maven-plugin/using-svnjava.html

SVNのリビジョン番号をそのまま埋め込むタイプ：

  • maven-svn-revision-number-plugin - Maven plugin that retrieves the revision number and the status of the Subversion working copy directory - Google Project Hosting
    • http://code.google.com/p/maven-svn-revision-number-plugin/
  • Maven SVN Revision Number Plugin - Introduction
    • http://maven-svn-revision-number-plugin.googlecode.com/svn/site/index.html
  • svn - How to get revision number from subversion using maven? - Stack Overflow
    • http://stackoverflow.com/questions/654572/how-to-get-revision-number-from-subversion-using-maven
  • svn - How to fetch current subversion revision number and URL with maven - Stack Overflow
    • http://stackoverflow.com/questions/750591/how-to-fetch-current-subversion-revision-number-and-url-with-maven
  • java - How to inject SVN revision number into JAR? - Stack Overflow
    • http://stackoverflow.com/questions/3914477/how-to-inject-svn-revision-number-into-jar
  • Maven - Cookbook - How To Add SVN Revision To A JAR Manifest
    • http://maven.apache.org/plugin-developers/cookbook/add-svn-revision-to-manifest.html

  • java - How to read MANIFEST.MF inside WAR application? - Stack Overflow
    • http://stackoverflow.com/questions/4239368/how-to-read-manifest-mf-inside-war-application
